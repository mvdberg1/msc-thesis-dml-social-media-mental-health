# Canvas and GitHub submission checklist

## Files to submit on Canvas

- `thesis.pdf`: final thesis.
- GitHub repository link, if Canvas allows a URL.
- If Canvas requires files rather than a link, upload a zip of the repository
  after checking that it does not contain restricted UKHLS data.

## Before uploading to GitHub

1. Confirm that `UKDA-6614-stata/` is not included.
2. Confirm that no `.dta` files are included.
3. Confirm that cleaned individual-level files under `data/analysis/` are not
   included.
4. Confirm that `README.md` and `REPLICATION.md` explain how a licensed user
   can reproduce the analysis.
5. Confirm that `thesis.pdf` opens and is the final version.

## Suggested GitHub repository description

Replication package for an MSc thesis on social media use, adolescent mental
well-being, panel models, and Double Machine Learning using Understanding
Society youth data.

## Suggested Canvas note

The submitted GitHub repository contains the thesis source files, code,
generated figures and tables, and replication instructions. The raw UKHLS data
and cleaned individual-level analysis files are not included because they are
restricted under the UK Data Service licence; replication requires users to
obtain their own licensed copy, as documented in `REPLICATION.md`.
